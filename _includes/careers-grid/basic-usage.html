Getting Started

To use the grid system, wrap the desired column widths within a .-row class. NOTE: Row and column classes will only work when placed within a .container class.

Making Rows

Rows are used to create groupings of content. You must declare a row so that all columns contained within it can have their widths properly set. For how to nest grids properly, please see "Nesting Columns".

NOTE: All content should be contained within in .-row class. It is recommended that content lives within column divs.

Making Columns

Columns are used to divide content within row groupings. Columns must be the immediate children of rows. If more than 12 columns are placed within a row, any .-col* group exceeding 12 columns will be wrapped onto a new line.
