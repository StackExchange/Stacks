--- layout: page title: Team identification description: Emails sent from a Team on Stack Overflow for Teams should be clearly labeled. ---

Team icon and name should be in a prominent, noticeable location in the email. The icon color should match the Team’s theme color as defined in their display settings (eg. https://stackoverflow.com/c/team-name/admin/settings/theme) on Stack Overflow or Stack Overflow for Enterprise.

If space is tight, an icon-only version can be used.

{% tip %} Note: An email should only contain content from one Team. A Team’s content should never appear mixed alongside with content from another Team or public content in a single email. {% endtip %} {% header "h2", "Example" %}
