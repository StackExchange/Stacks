{% header h2 | Fluid cells %}

By default all child elements of a flex have a default value of flex: 1 0 auto;. This means that by default all cells will only be as wide as their content. If you would like a cell or all the cells to fill the remaining space, apply the .fl1 to the cell, or .flex__fl1 to the parent to apply to all children.

{% header h2 | Fixed cells %}

{% header h2 | Nested flexs %}

flexs can be nested within each other. This allows you to create unique, custom layouts without needing to create new, custom code or override existing styles.

{% icon Alert %}
Note: Nested flexs with gutter spacing will conflict with each other. If you are nesting a flex with gutter spacing into a flex that also has gutter spacing, the child's parent wrapper margins will be overwritten by the parent. To have the child flex gutter spacing honored, you must place the child flex within a .flex--cell wrapper first. This allows the parent and child flex gutter spacing to act correctly without interfering with each other.
