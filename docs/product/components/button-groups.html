This component is used in our questions view, and is frequently used in conjunction with other form elements such as search fields and sorting dropdowns. If the content you’re interacting with is a simple paring down or filter of the current view, it’s appropriate to use the .s-btn-group component. Add the class .is-selected and the aria-current attribute with the appropriate value to show the currently selected button. ---

{% header "h2", "Accessibility" %}

When used as navigation or a filter, button groups should include the aria-current attribute with the appropriate value.

Item Applied to Description
aria-current="[value]" .s-btn.is-selected When using a button group for navigation or filtering, include the aria-current attribute with the appropriate value to the selected button. Most commonly used values are page, step, and true.
