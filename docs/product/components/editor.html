{% header "h2", "Installation" %}

Because of its size, the Stacks editor is bundled independently of Stacks. You can install it a few ways:

{% header "h3", "NPM" %}

The Stacks Editor is available as an NPM package. To make it available in your node modules, npm install @stackoverflow/stacks-editor

{% header "h3", "Import via Modules or CommonJS" %}
{% highlight js %} import { StacksEditor } from "@stackoverflow/stacks-editor"; // Don’t forget to include the styles as well import "@stackoverflow/stacks-editor/styles.css"; new StacksEditor( document.querySelector("#editor-container"), "*Your* **markdown** here" ); {% endhighlight %}
