--- layout: page title: Spot illustrations description: Spot illustrations are the slightly grown up version of icons with a little more detail. They’re most often used in empty states and new product announcements. They’re built externally on the Icons repository. ---
{% header "h2", "Using the spots helpers" %}

Just like icons, spot illustrations are delivered as a Razor helper in Stack Overflow’s production environment, a custom liquid tag in our documentation’s Eleventy site generator, and a JavaScript helper for use in prototypes.

Helpers should be used for all spot illustrations in lieu of SVG sprite sheets or static raster image assets. This ensures a single source of truth for all spot illustrations.

{% header "h3", "Basic" %}
{% highlight html %} @Svg.Spot.Wave {% spot "Wave" %} {% endhighlight %}
{% spot "Wave" %}
{% header "h3", "Arbitrary classes" %}

Spot illustrations can be colored on the fly with support for arbitrary classes.

{% highlight html %} @Svg.Spot.Wave.With("fc-orange-400 float-right js-dropdown-target") {% spot "Wave", "fc-orange-400 float-right js-dropdown-target" %} {% endhighlight %}
{% spot "Wave", "fc-orange-400 float-right js-dropdown-target" %}
