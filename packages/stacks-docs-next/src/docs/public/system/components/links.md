    <p class="docs-copy">In this case, you can add the <code>s-anchors</code> class together with one of the modifiers
        <code>s-anchors__default</code>, <code>s-anchors__grayscale</code>, <code>s-anchors__muted</code>,
        <code>s-anchors__danger</code>, or <code>s-anchors__inherit</code> to the container.
