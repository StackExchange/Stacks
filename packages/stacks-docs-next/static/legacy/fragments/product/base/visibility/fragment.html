Atomic visibility classes allow you to quickly change an element’s visibility.

Classes

Section titled Classes
Class Definition
.v-visible The element visible
.v-visible-sr The element is visible only to screen readers
.v-hidden The element is invisible, but still affects layout as normal

Examples

Section titled Examples
<div class="v-visible">…</div>
<div class="v-visible-sr">…</div>
<div class="v-hidden">…</div>
