{% header "h2", "Basic style" %}

In its simplest form, .s-sidebarwidget is comprised of a .s-sidebarwidget--content section and optional .s-sidebarwidget--header and .s-sidebarwidget--footer sections. Together these classes create a widget with appropriate inner spacing for you to put whatever you want into it.

{% icon "Info" %} By default the content is a flex container. If you require display: block instead, add the d-block class.
{% icon "Alert" %} The examples of s-sidebarwidget--header are shown with h2 elements, but the appropriate heading level may differ depending on context. Please use the appropriate heading level for your context to ensure heading levels only increase by 1.
