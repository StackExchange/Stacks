<Title>Toast</Title>

To show and hide toasts you need to first ensure that a `Toaster` component is rendered in your page.<br />
Ideally only one `Toaster` component is rendered per page.

### showToast(message, options)

Returns a toast ID that can be used with `hideToast`.
