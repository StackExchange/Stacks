Using icons

Our icons are delivered via inline SVG. In order to not do this manually, a helper function has been built to include icons. Simply call the helper function with the icon of your choice, and it will automatically be inserted into your view.

Please use the helper functions in lieu of SVG sprite sheets, or raster image assets. This ensures there is one source of truth for our entire icon set.
